## 🚀 Overview
**NOPE!** is a modern Web Application Firewall (WAF) that protects small sites and startups from automated threats. We realized that 43% of cyber attacks target small businesses, yet most security tools require a DevOps team to manage. **NOPE!** solves this by providing enterprise-grade edge protection with a simple, human-centric interface.

### ✨ Key Features
- **AI-Driven Threat Analysis:** Uses Gemini 1.5 Flash to explain SQLi and XSS attempts in plain English.
- **Edge-Level Blocking:** Drops malicious traffic before it ever touches your server, saving you bandwidth and CPU.
- **ISO 27001 Readiness:** Generate one-click security compliance reports to show investors your site is secure.
- **Glass-Morphism UI:** A beautiful, intuitive dashboard built for founders and developers.
- **Persistent Auth:** Custom-designed "Remember Device" toggle for a seamless 30-day login experience.

## 🛠️ Technical Stack
- **Backend:** Python / Flask
- **Frontend:** Tailwind CSS / Lucide Icons
- **Intelligence:** Google Gemini API
- **Infrastructure:** Gunicorn / Cloud-ready (Render/GCP)
